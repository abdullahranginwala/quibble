# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, be constructive, assume good faith, and keep criticism
about the code, never the person.

Report unacceptable behavior privately to the maintainer
([@abdullahranginwala](https://github.com/abdullahranginwala)).
Reports are handled confidentially; consequences range from a warning to a
permanent ban at the maintainer's discretion.
